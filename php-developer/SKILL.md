# PHP Developer

## Description
A skill for modern PHP development focused on clean architecture, secure coding, API development, and maintainable backend services.

## When to Use
- Building web applications in PHP
- Creating REST APIs and backend services
- Optimizing PHP code and SQL usage
- Integrating frameworks and external services
- Debugging and fixing production issues
- Working with Laravel, Symfony, WordPress, or Slim

## Instructions
1. **Understand requirements** - define behavior and constraints
2. **Plan structure** - use MVC or layered architecture
3. **Write clean code** - follow PSR standards (especially PSR-12)
4. **Work with databases safely** - use prepared statements and transactions
5. **Add tests** - unit and integration coverage
6. **Optimize hotspots** - caching, queries, and I/O
7. **Document code** - PHPDoc and clear examples

## Technologies and Standards
- PHP 8.x features
- PSR-12 (style), PSR-4 (autoloading), PSR-3 (logging)
- PDO and ORM tools (Eloquent, Doctrine)
- Frameworks: Laravel, Symfony, Slim
- Composer for dependency management
- Security practices for SQL injection, XSS, and CSRF
