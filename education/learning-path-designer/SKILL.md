# Learning Path Designer

## Description
A general-purpose education skill for designing learner journeys, milestones, and progression strategies from beginner to advanced levels.

## When to Use
- You need to design a structured learning journey
- Topics must be ordered by prerequisite and difficulty
- You want clear milestones and checkpoints
- The audience has mixed skill levels
- Learning goals must be aligned with measurable outcomes

## Instructions
1. **Define learner profile** - current level, goals, constraints, and context
2. **Set learning outcomes** - what the learner should know or do after completion
3. **Map prerequisites** - prior knowledge, missing concepts, and dependencies
4. **Sequence topics** - order content from foundations to advanced material
5. **Insert milestones** - practice checkpoints, reviews, and mastery gates
6. **Balance pace and depth** - avoid overload while keeping progress continuous
7. **Recommend next steps** - follow-up topics, exercises, or assessments

## Tools and Methods
- Prerequisite mapping
- Skill progression ladders
- Milestone planning
- Learning outcome design
- Difficulty staging
